# Pencil-Prop
a Simple Pencil prop for CVR

#Controls
Desktop:  LMB = Writing
          Press E = Erase

VR:       Trigger = Writing
          Touch the Rubber for 2 seconds while holding the Pencil = Erase
          
#Dependencies
CCK 3.3+ 
Download: https://developers.abinteractive.net/cck/setup/
